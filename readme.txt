== Description ==

*A cool, beautiful method that allows themes to be previewed without activation.<br>
*It will generate a button on the page link, when clicked, will Xianchu theme switch panel<br>
*Ajax way through to get all of your theme, and then displayed on the screen, then you can switch the theme of free choice. <br>
*After the switch there will be cookie records. <br>
*You can customize the cookie is valid. <br>
*You can also customize the display of the link button location.<br><br>
*If you have questions, Please contact [imqiqiboy#gmail.com] or leave a message in my blog(http://www.qiqiboy.com).<br>
*这是一种很酷的，奇妙的主题切换预览方式<br>
*插件将会在你的页面上生成一个按钮链接（你也可以自定义，不用插件自己生成），点击后将会弹出一个面板<br>
*通过ajax请求，插件会获取到你当前主机上的所有可用主题，然后按顺序将这些主题显示在弹出的面板上。<br>
*你此时可以点击任意主题，就可以切换到你所选择的主题了。当你点击时，插件又会发送一条ajax请求，主题设置切换好后，页面将自动刷新，然后你就能看到一个全新的主题了<br>
*主题切换会记录cookie，你可以自定义cookie的有效期，在有效期内将会一直显示所切换的主题<br>
*你也可以自定义按钮连接的显示位置、显示样式。<br>
*提供技术支持及咨询，联系邮件请发【imqiqiboy#gmail.com】或者在我博客【http://www.qiqiboy.com】留言<br>

<div style="color:#ff0000;font-weight:bold;">
注意：请最好不要勾选“自定义显示位置”，那样会很麻烦，容易出错。一切都交给插件吧！<br>
你可以直接将要作为按钮的图片放到插件的“img”目录中，然后在后台就可以选择你添加的这张图片了。<br>
但是还要注意，如果你选择的图片的宽度和高度不是默认的22，那么你还需要在图片的宽度和高度选项中填写你的这张图片的宽度和高度，否则图片将无法完整显示。<br>
</div>
== Installation ==

1. Download the plugin archive and expand it (you've likely already done this).
2. Put the 'wave-Your-theme' directory into your wp-content/plugins/ directory.
3. Go to the Plugins page in your WordPress Administration area and click 'Activate' for Wave Your Theme.
4. Go to the Wave Your Theme Options page (Settings > Wave Your Theme Option).
